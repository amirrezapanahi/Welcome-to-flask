###  Flask Hello World

یک پروژه‌ی ساده با **Flask** که فقط پیام "Hello, World!" را در صفحه اصلی نمایش می‌دهد.

---

####  اجرا

1. **نصب وابستگی‌ها**

   ```bash
   pip install -r requirements.txt
   ```

2. **اجرای برنامه**

   ```bash
   python app.py
   ```

3. **باز کردن در مرورگر**

   ```
   http://127.0.0.1:5000/
   ```
